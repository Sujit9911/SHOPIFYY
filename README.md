# 🛍️ SHOPIFYY — E-Commerce Web Application

> A full-stack e-commerce web application built with **Spring Boot**, **JSP**, and **Maven**.

---

## 📌 About the Project

**SHOPIFYY** is a Java-based e-commerce web application developed using the Spring Boot framework. It follows a layered MVC architecture with Controllers, Services, DAOs, and Models. The application supports user authentication, product browsing, cart management, billing, and contact functionality — all rendered through JSP views.

---

## ✨ Features

- 🔐 **User Authentication** — Login & Registration system
- 🏠 **Home & Index Page** — Landing pages for public and private users
- 🛍️ **Product Listings** — Browse and view all available products
- 🛒 **Shopping Cart** — Add products and manage cart items
- 🧾 **Bill Generation** — View billing details after purchase
- 📞 **Contact Page** — Users can reach out via contact form
- ℹ️ **About Page** — Information about the platform
- 🔒 **Public & Private Headers** — Separate navigation for guests and logged-in users

## 🚀 Getting Started

### Prerequisites

- Java 17+
- Maven 3.x
- Any IDE (IntelliJ IDEA / Eclipse / VS Code)
- MySQL or any configured database

### Run the Project

```bash
# Clone the repository
git clone https://github.com/your-username/shopifyy.git

# Navigate to the project folder
cd shopifyy

# Run using Maven Wrapper
./mvnw spring-boot:run

# OR on Windows
mvnw.cmd spring-boot:run
```

The app will start at: `http://localhost:8080`
